🎓 Student Performance Prediction

An end-to-end Machine Learning project that predicts student academic performance using demographic and educational features, enabling early intervention and data-driven decision-making.

📌 Problem Statement

Educational institutions often lack systems to proactively identify students at risk of poor academic performance.

This project builds a predictive model to:

Forecast student scores
Identify key influencing factors
Assist educators in targeted interventions
🎯 Key Features
🔍 Exploratory Data Analysis (EDA)
🧹 Data preprocessing & feature engineering
🤖 Multiple ML models (Regression & Ensemble)
📈 Model evaluation & comparison
🌐 Interactive web app using Streamlit
📦 Modular and production-ready pipeline
📊 Dataset

The dataset includes the following features:

Gender
Race/Ethnicity
Parental Level of Education
Lunch Type
Test Preparation Course
Math, Reading, Writing Scores

📁 Source: Kaggle – Student Performance Dataset
